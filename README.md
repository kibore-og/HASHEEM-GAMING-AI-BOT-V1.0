## рҹ“Ӣ AMRI ZA BOT

| Amri | Maelezo | Mfano |
|------|---------|-------|
| `.ai [swali]` | Uliza AI swali lolote | `.ai habari za leo` |
| `.play [mchezo]` | Anza mchezo | `.play quiz` |
| `.sticker` | Geuza picha kuwa sticker | Tuma picha + `.sticker` |
| `.translate [lugha] [maneno]` | Tafsiri maneno | `.translate en habari` |
| `.help` | Orodha ya amri zote | `.help` |
| `.ping` | Angalia kama bot iko hai | `.ping` |
| `.info` | Maelezo ya bot | `.info` |

> рҹ’Ў Tuma `.help` kwenye kikundi au DM ya bot kupata orodha kamili ya amri zote!
